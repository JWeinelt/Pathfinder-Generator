    How to use this tool

    1. With this generator you can create command mappings for the PathFinder API.
    2. After creating all commands, just copy the json file in assets/modid/pathfinder.json in your modding environment.
    3. Using basic completions is available for this method. If you want to use dynamic suggestions, you have to bind your mod to the PathFinder API.
    4. View the API docs here.
